## 计划

- [x] 核对隐藏的 Human 战术台、实时协议、票据、回放和统一 rating 归因。
- [x] 定义手动参与的首发范围、交互、节奏和断线规则。
- [x] 将 Human 重新开放拆为训练 Beta、统一排位、人类对战和人机协同阶段。
- [ ] 确认首发是否接受“仅 Human 对平台 Agent 的训练 Beta”。

## 验收标准

- 新用户无需代码或 Agent Key 即可完成手动教学和训练赛。
- 手动操作使用点选/拖拽航线与兵力分配，不要求理解弧度参数。
- Human 与 Agent 使用同一规则和命令边界；训练不影响 rating。
- 无操作、断线、刷新、退出和过期命令均有确定行为。
- 所有记录显示控制标签，未来排位仍为一个榜单和唯一 rating。

## 验证

- 浏览器走通中英文创建完成页、教学、手动训练、断线恢复、结果和回放。
- 对 2.5 秒候选回合窗口执行真实可玩性测试，记录时长与命令过期率。
- 自动验证权限、私有观察、命令上限、step 归属、幂等、恢复和回放归因。
- 验证 Human feature flag 关闭后不会泄露入口或破坏 Agent 流程。

## 备注

- 当前仓库已实现 Human/Agent 共享协议、WebSocket gateway、战术台和控制标签；公开入口默认关闭。
- 当前 worker 环境默认回合窗口为 10 秒，不适合作为手动玩法的最终节奏。
- 当前控制类型只有 `human` 与 `agent`；人机协同需要新的归因语义，不应伪装成其中任一种。
- 先开放训练 Beta，可以在不影响统一 rating 的前提下验证真实可玩性和连接质量。
