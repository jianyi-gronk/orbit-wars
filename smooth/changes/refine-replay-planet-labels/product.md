# 产品需求

## 背景

公开回放当前在每颗星球旁显示“星球 ID · 当前兵力”。星球 ID 属于内部关联信息，对观看者没有帮助，而且外置标签会让密集战场显得杂乱、产生文本重叠。

## 目标

- 回放画面不展示星球 ID。
- 当前兵力直接居中显示在星球内部，并随权威帧同步更新。
- 星球 ID 继续保存在回放数据中，用于跨帧重建、事件关联和交互身份识别。

## 验收标准

- 公开回放中的每颗星球只显示取整后的当前兵力，不显示 ID 或分隔点。
- 兵力文本在星球圆形内部水平、垂直居中。
- 回放加载、播放、时间轴和事件关联行为不受影响。
- 实时战场默认展示保持兼容，不因本次回放视觉调整改变数据协议。
