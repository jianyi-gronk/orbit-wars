# 产品需求

## 背景

当前战场中央太阳使用虚线危险环、尖刺、日冕折线和多层小核心，视觉更像战术标记，既不自然，也偏离 Kaggle 参考中的简洁太阳表现。

## 目标

战场中央太阳应首先被识别为一颗具有实体体积和柔和光照的恒星：使用完整的大金色圆盘表达碰撞实体，使用宽而连续的暖色光晕融入深色宇宙背景。

## 范围

- 太阳实体尺寸继续对应现有规则中的 `SUN_RADIUS`，不改变碰撞规则、地图坐标和回放数据。
- 使用单一金黄色实体圆盘与约 2.5～3 倍实体半径的柔和径向光晕。
- 移除虚线危险环、放射刻度、尖刺、折线日冕和白色小核心。
- 实时战场和历史回放继续复用同一套太阳渲染。

## 验收标准

- 太阳在正方形地图中心以完整金黄色圆盘显示，尺寸明显大于旧版小核心。
- 光晕由太阳边缘向外连续、柔和衰减，不出现明显同心色带。
- 页面不再展示虚线危险环、尖刺、刻度或多层靶心式核心。
- 不改变太阳碰撞半径、星球位置、舰队轨迹、回放控制及中英文页面行为。
