## 验收标准

- 无 Agent Key 用户可以用 ready 模板直接完成训练赛。
- 站内支持调参和代码两种策略优化方式，草稿不会因切换视图丢失。
- 新舰队获得可配置的 30 次标准 AI 优化额度；普通任务扣 1 次，深度分析扣 2 次，失败不扣。
- AI 只能提出可审查补丁，不能自动写入、发布或参赛；额度耗尽不影响手动闭环。
- 候选先模拟验证，只有通过后才能发布不可变版本。
- 外部 Agent 保留为可选高级入口。
- 新增界面与反馈支持中英文和窄屏。

## 验证

- 覆盖无 Key、ready 模板、草稿恢复、模拟失败、模拟成功、发布、恢复旧版本等状态。
- 验证模拟不改变 rating、公开战绩或当前版本。
- 验证沙盒错误脱敏、模板许可证与草稿隐私。
- 验证 AI 上下文脱敏、明确同意、余额原子扣减、失败返还、日限额、并发限制和全站预算熔断。
- 浏览器走通中英文“创建 → 训练 → 回放 → 优化 → 模拟 → 发布 → 再战”。

## 备注

- 当前创建舰队时已自动配置 `platform-basic` 或 `kaggle-structured-v11` 的 ready 版本。
- 当前 Agent Key 是自动化 API 的授权凭证，不是服务器执行 ready 策略的技术前提。
- 第一阶段使用 `deepseek-v4-flash` 提供可选 AI 解释、建议和补丁；按当前官方价格，单次 32K 输入、8K 输出的缓存未命中上限约为 0.048 元，30 次理论上限约为 1.44 元/账户。
- 免费额度首发建议为 30 次标准优化、每日最多 5 次、单账户并发 1；具体数值保持服务端可配置。
- DeepSeek 官方支持 `user_id` 隔离；技术设计应使用站内匿名标识，不发送直接身份信息。

- 是否在免费额度耗尽后开放付费充值或用户自带模型 Key，留到后续商业化阶段决定。

- 本地没有 DeepSeek API Key 时实现完整降级与可注入测试，不伪造线上 AI 结果。
