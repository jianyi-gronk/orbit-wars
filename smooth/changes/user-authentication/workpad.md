- 邮箱认证基础保留但默认关闭；Google 暂不开放。
- 账户之间的舰队、策略、密钥和私有模拟严格隔离。
- 生产环境不再使用固定 `preview-commander`。
- 认证页面完整支持中英文、窄屏、键盘与安全错误反馈。
