# 产品需求

## 背景

回放目前只展示每帧的星球归属和驻军变化，未展示在途舰队。观看者会看到兵力突然减少或星球突然易手，却看不到攻击从哪里来、向什么方向飞行。

## 目标

- 回放按每一帧权威数据展示所有在途舰队。
- 每支舰队通过阵营色、移动朝向和短尾迹清晰表达飞行方向。
- 舰队当前携带兵力随舰标展示，不显示内部舰队 ID。
- 播放、逐帧和时间轴跳转时，舰队位置与星球状态同步更新。

## 范围

- 使用 replay 已保存的舰队位置与角度，不推断目标星球。
- 本次不修改引擎、回放格式或历史回放数据。
- 实时战场组件同时具备展示舰队的能力，但本次核心验收路径是公开回放。

## 验收标准

- 指定回放在有 `fleets` 的帧能看到蓝/红在途舰队、方向尾迹和携带兵力。
- delta 未提供舰队字段时继承上一帧，显式 `fleets: []` 时清空。
- 舰队位置按 `x/y` 映射到与星球相同的战场坐标系，朝向按 `angle` 绘制。
- STEP 0 或无舰队帧不出现幽灵舰队。
- 回放主播放按钮在 hover 和键盘 focus 状态下文案保持清晰可见。
- 既有回放加载、星球兵力显示、播放控制和中英文页面不退化。
