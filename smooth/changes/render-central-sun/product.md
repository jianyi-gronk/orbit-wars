# 产品需求

## 背景

规则引擎在战场中心 `(50, 50)` 存在半径为 `10` 的太阳碰撞区，舰队路径穿过该区域会被直接移除。当前 Web 战场只绘制同心网格，没有呈现太阳，导致观看者误以为兵力在地图中央凭空消失。

## 目标

- 在回放和实时战场中央明确渲染太阳。
- 太阳的危险边界对应规则中的半径 `10`，而不是纯装饰尺寸。
- 通过核心亮度、外层辉光和危险边界让太阳与普通星球明显区分。
- 不修改引擎、回放格式或既有碰撞规则。

## 验收标准

- 战场中心始终可见太阳，中心与规则坐标 `(50, 50)` 一致。
- 可见危险区域使用 `SUN_RADIUS = 10` 映射到当前战场缩放。
- 太阳绘制在背景网格之上、星球和舰队之下，不遮挡兵力数字与舰标。
- Web 测试、类型检查、lint 和生产构建通过。
